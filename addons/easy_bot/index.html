    EasyBot¶

    Note

    The following classes come from the scurrypy.addons.easy_bot module.

    EasyBot ¶

    Bases: Client

    Prepackaged interaction, prefix, and event convenience for most of your needs!

    Parameters:

    • token (str) –

      the bot's token

    • application_id (int) –

      the bot's user ID

    • intents (int, default: DEFAULT ) –

      gateway intents. Defaults to Intents.DEFAULT.

    • prefix (str, default: None ) –

      prefix if using prefix commands.

    • sync_commands (bool, default: True ) –

      Whether commands should be synced with changes. Defaults to True.

    prefixes = PrefixAddon(self, prefix) instance-attribute ¶

    commands = InteractionAddon(self, sync_commands) instance-attribute ¶

    bot_events = EventsAddon(self) instance-attribute ¶

    token = token instance-attribute ¶

    application_id = application_id instance-attribute ¶

    intents = intents instance-attribute ¶

    shards: list[GatewayClient] = [] instance-attribute ¶

    logger = logger or Logger() instance-attribute ¶

    events = {} instance-attribute ¶

    startup_hooks = [] instance-attribute ¶

    shutdown_hooks = [] instance-attribute ¶

    addons = AddonManager(self) instance-attribute ¶
